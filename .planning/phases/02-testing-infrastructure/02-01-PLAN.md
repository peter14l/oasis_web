Add Vitest configuration to vite.config.js:

1. Add reference types at the top:

```javascript
test: {
  globals: true,
  environment: 'jsdom',
  setupFiles: ['./src/test/setup.js'],
}
```

This enables:
- globals: Makes describe, it, expect available globally
- jsdom: Browser-like DOM environment
- setupFiles: Runs setup before tests
